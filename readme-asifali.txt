##How to run in your machine##

//Clone the repository 
    Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac) to open the command palette.
   Type "Git: Clone" and select it. 
   Enter the URL "https://github.com/aliasif72/fyle-internship-challenge-23.git"
   Choose a local folder where you want to clone the repository.

// Navigate to folder
   Run this command in your terminal after cloning
   cd fyle-internship-challenge-23

//Node Version
   Install Node as i have built this project using node v20.11.0.
 
// Install all the dependencies
   Run npm install in your terminal.

// Launch the application
    Run npm start in your terminal

// For running test cases 
   use npm run test in your terminal

//How to run the application
  Open any browser(mine:chrome) and navigate to http://localhost:4200/
  A search box will appear with a submit button.
  Enter a github username and hit submit.
  You will see his/her profile and public repositories with a default profile photo i used.
